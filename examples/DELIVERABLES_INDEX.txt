TECHNICAL SPECIFICATIONS
═══════════════════════════════════════════════════════════════════════════

Language: Rust
Framework: Native Rust (no external dependencies for main tool)
Data Format: In-memory KBO structures
Output: Formatted text report
Complexity: O(n log n) due to sorting
Memory: ~1 MB for 28 objects

Algorithm:
  1. Load KBO objects with semi-major axis data
  2. Sort by semi-major axis
  3. Calculate gap between each consecutive pair
  4. Filter gaps >20 AU with inner boundary >50 AU
  5. Calculate gap center (estimated planet location)
  6. Estimate mass: M = 5.0 × sqrt(gap_width / 20)
  7. Generate report with statistics

Data Source:
  - NASA/JPL Small-Body Database (SBDB)
  - 28 select Trans-Neptunian Objects
  - Semi-major axis values (AU)
